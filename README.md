Шаг 1: Установите Docker Desktop
Скачайте Docker Desktop с официального сайта: https://www.docker.com/products/docker-desktop/

Установите как обычную программу

Шаг 2: Скачайте файлы программы
Скачай архив
Распакуйте скачанный архив в любую папку на компьютере

Шаг 3: Запустите программу
Откройте Командную строку или PowerShell:

Windows: нажмите Win + R, введите cmd, нажмите Enter

macOS: откройте Терминал из папки "Утилиты"

В командной строке перейдите в папку с программой:

bash
cd путь_к_вашей_папке
Например:

bash
cd C:\Users\Имя\Downloads\admission-analyse-main

Запустите установку одной командой:

bash
docker-compose up -d
